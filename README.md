# CodeAlpha EDA Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic Dataset to identify patterns, trends, and factors affecting passenger survival.

## Objectives
- Understand dataset structure
- Analyze missing values
- Explore survival trends
- Create visualizations
- Draw conclusions

## Tools Used
- Python
- Pandas
- Matplotlib
